Purpose:

Product Class: Object with fields on what defines a Product

ProductInventory Class: To manage lists on Products with added methods

Order Class: Object with fields and methods for management, defining what an Order is

OrderItem Class: Object on what product will be ordered, along with getter/setter to help on
another class, along with calculating totals

OrderManager Class: Manages Order lists, for finding product Id, updating statuses, canceling,
adding, printing

VectorComparisonDemo Class: To showcase performance differences with vector and arraylist along with memory
usage difference

VectorUtils Class: Help generalize methods, since it is generics, we can use the methods made on any type arrays

GenericContainer Class: Adds utility methods such as adding, getting, size for Vectors on any type
